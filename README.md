Crowdfunding DApp.

To run this project: You will need React, web3.js, hardhat, metamask
1. 'npm run dev' on frontend/crowdfundingDApp
2. 'npx hardhat node' on backend folder to start hardhat node
3. 'npx hardhat run scripts/deploy.js --network localhost' on backend folder to deploy contracts

This project was partially completed following Thirdweb's tutorial: https://www.youtube.com/watch?v=GxFXHXxd83E&list=LL&index=1&t=6468s
